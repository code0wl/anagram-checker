# A module for generating anagrams

### Installation

#### npm
```bash
$ npm i anagram-generator -S
```

#### yarn
```bash
$ yarn add anagram-generator
```

### Usage
```ecmascript 6
const anagramChecker = require("anagram-checker");

new anagramChecker('neat'); // ante, neat
```
